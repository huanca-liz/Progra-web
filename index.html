A través de diferentes páginas interactivas, el usuario podrá ingresar datos, visualizar resultados y comprender cómo funcionan estos algoritmos mediante el uso de HTML, CSS y JavaScript.

Además, se incluye una sección donde se combinan ambos conceptos, permitiendo observar cómo algunos números de la secuencia de Fibonacci también pueden ser números primos.

Este proyecto no solo busca mostrar resultados, sino también ayudar a entender la lógica detrás de cada algoritmo, fomentando el aprendizaje práctico y el desarrollo de habilidades en programación web.

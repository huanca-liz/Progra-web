## Progra web
## Numeros primos y Fibonacci
### Descripción

Este proyecto consiste en una página principal que integra tres secciones diferentes relacionadas con secuencias y cálculos matemáticos:

1. **Números Primos**: Permite identificar o generar números primos.
2. **Serie de Fibonacci**: Permite calcular y mostrar términos de la sucesión de Fibonacci.
3. **Combinación**: Integra funcionalidades de las dos secciones anteriores en una sola página.

Cada sección cuenta con sus propios archivos de estilos (**CSS**) y scripts (**JavaScript**) para mantener una organización clara y facilitar el mantenimiento del código.

### Estructura del Proyecto

```text

│

└── combinacion/
    ├── combinacion.html
    ├── combinacion.css

```

### Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript

### Funcionalidades

#### Números Primos

* Verificación de números primos.
* Generación de secuencias de números primos.

#### Fibonacci

* Cálculo de términos de la sucesión de Fibonacci.
* Visualización de la secuencia generada.

#### Combinación

* Integración de las funcionalidades de números primos y Fibonacci.
* Interfaz unificada para realizar operaciones relacionadas.

### Cómo Ejecutar el Proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en cualquier navegador web.
3. Seleccionar la sección deseada desde la página principal.
